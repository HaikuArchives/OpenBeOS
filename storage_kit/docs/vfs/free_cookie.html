free_cookie -- Free file resources


Called to free the resources allocated when a file is initially opened with a call to open. This function must be called once and only once after all calls to open have been matched with calls to close, and should free all the resources related to the file.

Declaration

int free_cookie (     void  * pVolume,
  void  * pNode,
  void   pCookie   );

Parameters

Input:
pVolume -- Pointer to the file system specific data structure returned by mount
pNode -- Pointer to the node for given file
pCookie -- Pointer for the cookie of the given file

Return

B_OK -- The function returned successfully

Notes
