fsync -- Flush all buffers of a file to disk


Forces a filesystem to write all cached data for a given file to disk

Declaration

int fsync (     void  * pVolume,
  void  * pNode   );

Parameters

Input:
pVolume -- File system specific volume structure returned by mount
pNode -- Pointer to the node of the file

Return

B_OK -- The function returned successfully

Notes

This description derived from the sources for OpenBFS and AtheosFS, as well as parts of Practical Filesystem Design by Dominic Giampoalo
