op_read_indexdir - Reads the next entry(s) in the attribute set.

Reads the next entry(s) in the index, and returns them as a dirent. The number of dirent's is specified by num.

Declaration

int op_read_indexdir(void *ns, void *cookie, long *num, struct dirent *buf, size_t bufsize);

Parameters

Inputs

ns - pointer to a volume structure
cookie - The index's cookie
bufize - Ignored by implementations seen so far

Outputs

num - The number of attributes returned in buf, can be zero.
buf - The dirents.

Return

B_OK - Operation completed
ENOMEM - Not enough memory
EINVAL - Invaild parameters
B_BAD_VALUE - Invaild parameters

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com) and OpenBeFS from the OpenBeOS CVS.
