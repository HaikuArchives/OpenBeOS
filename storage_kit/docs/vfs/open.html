open -- Open a File


Opens a file and returns a file cookie

Declaration

int open (   void  * pVolume,
  void  * pNode,
  int nMode,
  void  ** ppCookie   );

Parameters

Input:
pVolume -- volume on which the file resides
pNode -- vnode of the file to be opened
nMode -- read, write, read/write

Output:
ppCookie -- file cookie (i.e. file descriptor) to be used for all other file operations

Return

B_OK - Success

Notes

This description derived from the sources for OpenBFS and AtheosFS, as well as parts of Practical Filesystem Design by Dominic Giampoalo
