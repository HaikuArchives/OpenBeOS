op_open_query - Opens a query.

Creates a query based on the query string query, returning a query cookie.

Declaration

int op_open_query(void *ns, const char *query, ulong flags, port_id port, long token, void **cookie);

Parameters

Inputs

ns - pointer to a volume structure
query - The query string to execute
flags - ?
port - ?
token - ?

Outputs

cookie - The query cookie for use

Return

B_OK - Operation completed
ENOMEM - Not enough memory
EINVAL - Invaild parameters
ENOSPC - No space on device.
