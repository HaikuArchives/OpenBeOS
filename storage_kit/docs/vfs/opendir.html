opendir - Open a Directory


Opens a directory for reading

Declaration

int opendir ( void* pVolume, void* pNode, void** ppCookie );

Parameters

Input:
pVolume - pointer to a volume structure
pNode - pointer to a valid node structure representing a file on the volume

Output:
ppCookie - 'magic cookie' to be used for other directory calls

Return

B_OK - Operation completed
ENOMEM - Not enough memory
ENOTDIR - node is not a directory

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
