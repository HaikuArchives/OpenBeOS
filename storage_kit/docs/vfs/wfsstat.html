wfsstat -- Write statistical information about a volume


Updates a volume with data from a file system stat structure

Declaration

int rstat (     void  * pVolume,
  struct   fs_info  * psInfo
  long   nMask   );

Parameters

Input:
pVolume -- File system specific volume structure returned by mount
psInfo -- Pointer to a file system stat structure with which to update the volume
nMask -- Bitmask specifying which attributes to modifiy

Return

B_OK -- The function completed successfully

Notes

This description derived from the sources for OpenBFS and AtheosFS, as well as parts of Practical Filesystem Design by Dominic Giampoalo
