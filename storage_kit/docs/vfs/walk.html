walk -- Convert a filename into a Vnode

Translates a relative pathname and context into a full pathname and vnode

Declaration

int walk (   void  * pVolume,
  void  * pParent,
  const   char  * pzName,
  char  ** ppzNewPath,
  vnode_id  * pnVnodeNum   );

Parameters

Input:
pVolume - pointer to a volume structure
pParent - context of parent (structure of context is dependant on thefile system)
pzName - relative file name

Output:
ppzNewPath - pointer to a char* which will point to the full path name after the call
pnVnodeNum - vnode number of file

Return

B_OK - Operation completed
ENOENT - No such file
ENOMEM - Not enough memory to complete operation

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
