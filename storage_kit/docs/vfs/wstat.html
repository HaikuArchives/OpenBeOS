wstat -- Write statistical information about a file


Updates a file with data from a POSIX style stat structure

Declaration

int rstat (     void  * pVolume,
  void  * pNode,
  struct   stat  * psStat
  long   nMask   );

Parameters

Input:
pVolume -- File system specific volume structure returned by mount
pNode -- Pointer to the node of the file
psStat -- Pointer to a POSIX style stat structure to use for updating the file's statistical information
nMask -- Bitmask specifying which attributes to modifiy

Return

B_OK -- The function completed successfully

Notes

This description derived from the sources for OpenBFS and AtheosFS, as well as parts of Practical Filesystem Design by Dominic Giampoalo
