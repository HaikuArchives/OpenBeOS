read_vnode - Read a Vnode

Reads an inode from the disk and convert it into a vnode

Declaration

int read_vnode( void* pVolume, vnode_id nVnodeNum, char r, void** ppNode );

Parameters

Input:
pVolume - pointer to a volume structure
nVnodeNum - vnode to read
r - ?? (Not used in code I looked at)

Output:
ppNode - Pointer to a pointer to a node structure.

Return

B_OK - Operation completed
ENOMEM - Not enough memory to complete operation

Notes

read_vnode will allocate a node structure and pass it back in ppNode

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
