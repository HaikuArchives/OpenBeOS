setflags -- Set file open mode flags


Used to change the status of a file's open mode a la the POSIX fnctl() call

Declaration

int setflags (     void  * pVolume,
  void  * pNode,
  void  * pCookie,
  int   nFlags   );

Parameters

Input:
pVolume -- Pointer to the file system specific volume structure returned by mount
pNode -- Pointer to the node of the file
pCookie -- Pointer to the cookie for the file
nFlags -- Mode flags to set

Return

B_OK -- The function returned successfully

Notes

This description derived from the sources for OpenBFS and AtheosFS, as well as parts of Practical Filesystem Design by Dominic Giampoalo
