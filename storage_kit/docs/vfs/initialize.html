initialize -- create an initial filesystem on a device


Creates and initializes the data structures for a fresh filesystem on a device

Declaration

int initialize (   const   char  * pzDevName,
  void  * pParms,
  size_t   nLen   );

Parameters

Input:
pzDevName -- Pointer to a string containing the name of the device to initialize
pParms -- Pointer to a structure with parameters for initialization
nLen -- Length of of the structure pointed to by pParms

Return

B_OK -- The function returned successfully

Notes

This description derived from the sources for OpenBFS and AtheosFS, as well as parts of Practical Filesystem Design by Dominic Giampoalo
