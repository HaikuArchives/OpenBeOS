close - Close a file


Closes a file previously opened with open()

Declaration

int close ( void *pVolume, void *pNode, void *pCookie );

Parameters

Input:
pVolume -- volume to which the file belongs
pNode -- vnode of the file to close
pCookie -- cookie of the file to close

Return

B_OK - Success

Notes

This description derived from the sources for OpenBFS and AtheosFS, as well as parts of Practical Filesystem Design by Dominic Giampoalo
