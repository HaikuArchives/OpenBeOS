create - Create a file

Create a file

Declaration

int create( void* pVolume, void* pParent, const char* pzName, int nMode, int nPerms, node_id* pnVnodeID, void** ppCookie );

Parameters

Input:
pVolume - pointer to a volume structure
pParent - context of parent in which file is to be created
pzName - name of file
nMode - Open mode (e.g. O_EXCL, O_TRUNC)
nPerms - Permissions mask (rwx)

Output:
pnVnodeID - pointer to returned vnode ID
ppCookie - cookie that can be used to access the file

Return

B_OK - Operation completed
EEXIST - File already exists (and O_EXCL is set)
ENOMEM - Not enough memory to complete the operation
ENOENT - Parent doesn't exist

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
