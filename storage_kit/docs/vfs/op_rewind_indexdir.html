op_rewind_indexdir - Returns the indexdir to the first entry in the directory tree.

Returns the indexdir to the first entry in the directory tree

Declaration

int op_rewind_indexdir(void *ns, void *cookie);

Parameters

Inputs

ns - pointer to a volume structure
node - pointer to a valid node structure representing a file on the volume

Return

B_OK - Operation completed
ENOMEM - Not enough memory
EINVAL - Invaild parameters
B_BAD_VALUE - The index dir does not exist.

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
