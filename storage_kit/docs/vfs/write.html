write - Write to a file


Writes a number of contiguous bytes to a file

Declaration

int write (     void  * pVolume,
  void  * pNode,
  void  * pCookie,
  off_t   nPos,
  const   void  * pBuffer,
  size_t  * pnLen   );

Parameters

Input:
pVolume -- Pointer to the filesystem specific volume structure returned by mount
pNode -- Pointer to the node of the destination file
pCookie -- Pointer to the cookie for destination file
nPos -- Offset (in bytes) from beginning of file at which writing should begin

Output:
pBuffer -- Address of an array of bytes to be written

Input/Output:
pnLen -- [input] Pointer to the length (in bytes) of the array pointed to by pBuffer; [output] Pointer to the number of bytes actually written to the file

Return

B_OK -- The function completed successfully
ENOMEM -- Insufficient memory available to complete the operation
EINVAL -- Invalid input, or invalid on-disk data structure

Notes
