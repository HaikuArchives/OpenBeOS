writev -- vector read


Performs a scattered write to the given file from an array of iovec structures: 
iovec {
  void *pData;   // Pointer to a chunk of memory to read to or write from
  int  nLen;     // Length of said chunk of memory
}

Declaration

int readv (     void  * pVolume,
  void  * pNode,
  void  * pCookie,
  off_t   nPos,
  const   iovec  * pIOVector,
  size_t   nSize,
  size_t  * pnLength   );

Parameters

Input:
pVolume -- Pointer to the file system specific volume structure returned by mount
pNode -- Pointer to the node of the file
pCookie -- Pointer to the cookie for the file
nPos -- Offset from the beginning of the file at which to begin writing
nCount -- Number of bytes to write (?)

Output:
pIOVector -- Pointer to an array of iovec structures from which to write data to the file
pnLen -- [input] Pointer to the length (in bytes) of the array pointed to by pIOVec (?); [output] Pointer to the number of bytes actually written from the iovec structures pointed to by pIOVector

Return

B_OK -- The function returned successfully
