op_open_attrdir - Opens an attribute set.

Opens an attribute set for a given file, returning a cookie for future access.

Declaration

int op_open_attrdir(void *ns, void *node, void **cookie);

Parameters

Inputs

ns - pointer to a volume structure
node - pointer to a valid node structure representing a file on the volume

Outputs

cookie - The cookie for the attribute set.

Return

B_OK - Operation completed
B_NO_MEMORY - Not enough memory
ENOMEM - Not enough memory
EINVAL - Invaild parameters
ENOENT - The file does not exist.

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com) and OpenBeFS from the OpenBeOS CVS.
