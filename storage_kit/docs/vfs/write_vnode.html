write_vnode - Write a Vnode to disk

Prepares a vnode to be written to disk

Declaration

int write_vnode( void* pVolume, void* pNode, char r );

Parameters

Input:
pVolume - pointer to a volume structure
pNode - Pointer to a node structure.
r - ?? (Not sure - something to do with locking)

Return

B_OK - Operation completed
EINVAL - Invalid parameter
ENOMEM - Not enough memory to complete operation

Notes

This routine is called by the kernel when a vnode's reference count falls to 0 (i.e. it is no longer required) - it is responsible for freeing up all resources used by the vnode.

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
