op_remove_attr - Removes an attribute

Removes an attribute.

Declaration

int op_remove_attr(void *ns, void *node, const char *name);

Parameters

Inputs

ns - pointer to a volume structure
node - pointer to a valid node structure representing a file on the volume
name - The name of the attribute you want gone.

Return

B_OK - Operation completed
NOENT - The entry does not exist.

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com) and OpenBeFS from the OpenBeOS CVS.
