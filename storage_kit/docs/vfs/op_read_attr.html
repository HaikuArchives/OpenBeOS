op_read_attr - Read an attribute

Reads an attribute from disk

Declaration

int op_read_attr(void *ns, void *node, const char *name, int type, void *buf, size_t *len, off_t pos);

Parameters

Inputs

ns - pointer to a volume structure
node - pointer to a valid node structure representing a file on the volume
name - The attribute name you wish to read
pos - The offset into the attribute from which you wish to read

Outputs

buf - The returned attribute data.
len - The length of the attribute data

Return

B_OK - Operation completed
ENOMEM - Not enough memory
EINVAL - Invaild parameters
ENOENT - The file to which the attribute is part of does not exist.

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com) and OpenBeFS from the OpenBeOS CVS.
