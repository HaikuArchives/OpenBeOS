mount - Mount a volume


Mounts a device and returns its volume structure and the vnid of its root directory

Declaration

int mount (     nspace_id   nVolumeID,
  const   char  * pDevName,
  ulong   nFlags,
  void  * pParms,
  size_t   nLen,
  void  ** ppVolume,
  vnode_id  * pnRootNodeID   );

Parameters

Input:
nVolumeID - ID assigned to this volume by the kernel
pDevName - name of the device to be mounted
nFlags - read, write, read/write
pParms - paramters structure from the kernel
nLen - length of the structure pointed to by pParms

Output:
ppVolume - file system specific volume structure used for all other file system operations
pnRootNodeID - vnode id of the volume's root directory

Return

B_OK - Success
