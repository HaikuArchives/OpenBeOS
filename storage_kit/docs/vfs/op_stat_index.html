op_stat_index - Returns information on an index.

Returns information (index_info - type, size, last modified and creation date) on a named index.

Declaration

int op_stat_attr(void *ns, void *node, const char *name, struct attr_info *buf);

Parameters

Input

ns - pointer to a volume structure
node - pointer to a valid node structure representing a file on the volume
name - The index you want information about.

Output

buf - The returned index_info

Return

B_OK - Operation completed
ENOMEM - Not enough memory
B_BAD_VALUE - Invaild Parameters

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
