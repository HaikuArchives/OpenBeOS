sync -- Flush all buffers of a volume to disk


Forces a filesystem to write all cached data to disk

Declaration

int sync (     void  * pVolume,   );

Parameters

Input:
pVolume -- File system specific volume structure returned by mount

Return

B_OK -- The function returned successfully
