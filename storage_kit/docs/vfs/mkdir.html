mkdir - Create a directory

Create a directory

Declaration

int mkdir( void* pVolume, void* pParent, const char *pzName, int nMode );

Parameters

Input:
pVolume - pointer to a volume structure
pParent - context of parent in which file is to be created
pzName - name of file
nMode - Access mode (rwx)

Return

B_OK - Operation completed
EINVAL - Invalid parameter
ENOMEM - Not enough memory to complete operation
EEXIST - Directory already exists

Notes

This definition was devised using the source for the AtheOS file system driver (downloadable from bebits.com)
